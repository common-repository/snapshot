Adds the shortcode for snapshots.

== Description ==

Use `[snap url="http://google.com" alt="My description" w="400" h="300"]`

== Installation ==

1. Upload `snapshot` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy :)
